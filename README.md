# 入院費概算シミュレーター

Googleスプレッドシートを毎回読み込み、条件を順に選択して入院費概算を表示する静的HTMLサイトです。

## ファイル構成

- `index.html`: 画面本体
- `styles.css`: スタイル
- `app.js`: スプレッドシート読込・条件分岐・結果計算

## 使い方

1. `index.html` をブラウザで開く
2. 条件を順番に選択する
3. 最終画面で `月額上限 + 食事代` の概算合計を確認する

## 重要: スプレッドシートの公開設定

クライアント側(JavaScript)で直接読み込むため、対象シートは閲覧可能である必要があります。

1. Googleスプレッドシートを開く
2. 右上の「共有」をクリック
3. 「リンクを知っている全員」に変更
4. 権限を「閲覧者」に設定

## シートID / GID の変更

別シートに切り替える場合は、`app.js` の以下を更新してください。

```js
const SHEET_ID = "スプレッドシートID";
const SHEET_GID = "シートGID";
```
